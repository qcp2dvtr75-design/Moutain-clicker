# Горный Кликер (Мини-игра)
Простая игра на JavaScript, где нужно добывать энергию кликом по горе.

### Особенности:
* Система прокачки (авто-кирка и супер-бур).
* Анимация клика и звуковое сопровождение.
* Сохранение прогресса в браузере.
* Скрытый промокод "HACKER".
